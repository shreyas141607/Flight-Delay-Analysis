# Flight Delay Analysis Dashboard

## Overview
This project analyzes flight operations data to identify delay patterns, airport performance variations, and operational bottlenecks. Python was used for data cleaning and exploratory data analysis (EDA), while Power BI was used to build an interactive dashboard.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Power BI
- DAX
- Excel

## Key Features
- Flight delay trend analysis
- Airport performance comparison
- Delay distribution analysis
- Interactive KPIs and visualizations
- Data modeling and DAX measures

## Dashboard Insights
- Identified peak delay periods
- Analyzed airport performance metrics
- Evaluated delay distribution patterns
- Monitored operational trends

## Project Objective
Transform raw flight data into actionable insights and support data-driven decision-making through interactive visualizations.
